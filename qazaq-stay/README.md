# 🏨 Qazaq Stay — Сервис бронирования отелей в Казахстане

Современный полнофункциональный сайт для бронирования отелей по всему Казахстану.
Дизайн вдохновлён Booking.com и Airbnb. Технологии: **HTML5 + CSS3 + JavaScript + PHP + MySQL**.

---

## ✨ Возможности

- 🔍 Поиск отелей по городу, дате, количеству гостей
- 🌗 Светлая и тёмная тема (с сохранением выбора в localStorage)
- 👤 Регистрация, авторизация, личный кабинет
- 📅 Полноценное бронирование с расчётом стоимости и сервисного сбора
- ⚙️ Панель администратора: управление отелями, бронированиями, пользователями
- 📱 Полностью адаптивный дизайн (ПК, планшет, смартфон)
- 🎨 Анимации, hover-эффекты, плавные переходы
- 🔒 Безопасность: PDO + prepared statements, BCRYPT хеширование паролей, защита от XSS

---

## 📂 Структура проекта

```
qazaq-stay/
├── index.php             ← Главная страница (баннер, поиск, топ отелей, города)
├── hotels.php            ← Каталог отелей с фильтрами (город, цена, рейтинг)
├── booking.php           ← Страница бронирования (требует авторизации)
├── login.php             ← Авторизация
├── register.php          ← Регистрация
├── logout.php            ← Выход
├── profile.php           ← Личный кабинет (мои бронирования, статистика)
├── setup.php             ← Одноразовый скрипт для создания админа
├── db.php                ← Подключение к MySQL (PDO) + сессии + хелперы
├── style.css             ← Все стили (светлая/тёмная тема, адаптив, анимации)
├── script.js             ← JS: тема, мобильное меню, валидация форм, анимации
├── admin/                ← Панель администратора
│   ├── index.php         ← Дашборд (статистика, последние бронирования)
│   ├── hotels.php        ← Список отелей с управлением
│   ├── hotel_add.php     ← Добавление нового отеля
│   ├── hotel_edit.php    ← Редактирование отеля
│   ├── bookings.php      ← Все бронирования (подтвердить/отменить)
│   └── users.php         ← Список пользователей со статистикой
└── sql/
    └── qazaq_stay.sql    ← База данных + ~34 демо-отеля по 14 городам
```

### Что делает каждый файл

| Файл | Назначение |
|------|-----------|
| **db.php** | Создаёт PDO-подключение к БД, запускает сессию. Содержит хелперы `isLoggedIn()`, `isAdmin()`, `getCurrentUser()`, `e()` (защита от XSS) |
| **index.php** | Главная: навбар, hero-баннер с формой поиска, популярные города, топ-8 отелей по рейтингу, секция преимуществ |
| **hotels.php** | Каталог с фильтрами по городу/цене/рейтингу, сортировкой и пагинацией |
| **booking.php** | Форма бронирования: расчёт ночей × цена + 5% сервисный сбор, запись в БД |
| **profile.php** | Личный кабинет: данные пользователя, статистика, список бронирований с возможностью отмены |
| **login.php / register.php** | Регистрация (BCRYPT) и вход с проверкой через `password_verify` |
| **setup.php** | Создаёт админа `admin@qazaqstay.kz` / `admin123`. **Удалить после использования!** |
| **style.css** | CSS-переменные для тем, сетки, карточки отелей, формы, адаптив (брейкпоинты 1024px и 768px) |
| **script.js** | Переключение темы, мобильное меню, валидация дат, расчёт цены в реальном времени, IntersectionObserver для анимаций |
| **admin/*** | Полная CRUD-панель администратора |

## 🚀 Установка и запуск через XAMPP

### Шаг 1. Установите XAMPP

Скачайте с официального сайта: https://www.apachefriends.org/
Установите как обычное приложение. По умолчанию ставится в `C:\xampp` (Windows) или `/Applications/XAMPP` (macOS).

### Шаг 2. Скопируйте проект

Распакуйте архив и поместите папку `qazaq-stay` в:

- **Windows:** `C:\xampp\htdocs\qazaq-stay\`
- **macOS:** `/Applications/XAMPP/htdocs/qazaq-stay/`
- **Linux:** `/opt/lampp/htdocs/qazaq-stay/`

### Шаг 3. Запустите Apache и MySQL

Откройте **XAMPP Control Panel** и нажмите **Start** напротив:
- ✅ **Apache**
- ✅ **MySQL**

### Шаг 4. Создайте базу данных

1. Откройте в браузере: **http://localhost/phpmyadmin**
2. Слева нажмите **«Создать БД»** (New)
3. Введите имя: `qazaq_stay`
4. Сравнение (collation): `utf8mb4_unicode_ci`
5. Нажмите **Создать**

### Шаг 5. Импортируйте таблицы и демо-данные

1. Кликните на созданную БД `qazaq_stay` в левом меню
2. Перейдите во вкладку **Импорт** (Import)
3. Нажмите **«Выберите файл»** → выберите `sql/qazaq_stay.sql`
4. Нажмите **Импорт** (Go) внизу страницы
5. Должно появиться сообщение: «Импорт успешно завершён»

База создаст **3 таблицы**: `users`, `hotels`, `bookings`, и загрузит **34 демо-отеля** в 14 городах Казахстана.

### Шаг 6. Проверьте настройки подключения

Откройте `db.php` и убедитесь, что параметры подходят (стандартные для XAMPP):

```php
$host = 'localhost';
$dbname = 'qazaq_stay';
$username = 'root';
$password = '';  // У XAMPP по умолчанию пароль пустой
```

### Шаг 7. Создайте администратора

1. Откройте: **http://localhost/qazaq-stay/setup.php**
2. Скрипт создаст админа с данными:
   - **Email:** `admin@qazaqstay.kz`
   - **Пароль:** `admin123`
3. ⚠️ **Обязательно удалите файл `setup.php` после использования** — это требование безопасности!

### Шаг 8. Запустите сайт

Откройте в браузере:

🌐 **http://localhost/qazaq-stay/**

Готово! Сайт работает.

## 🛠️ Возможные проблемы

**❌ «Подключение к базе данных не удалось»**
→ Запустите MySQL в XAMPP Control Panel. Проверьте имя БД в `db.php`.

**❌ Порт 80 занят (Apache не стартует)**
→ Skype/IIS могут блокировать порт. В XAMPP → Apache → Config → httpd.conf замените `Listen 80` на `Listen 8080`, тогда сайт будет на `http://localhost:8080/qazaq-stay/`.

**❌ Кириллица отображается «крякозябрами»**
→ Убедитесь, что при импорте SQL используется кодировка `utf8mb4`. Файл `qazaq_stay.sql` уже в этой кодировке.

**❌ Не могу войти как админ**
→ Запустите `setup.php` один раз, затем удалите файл. Используйте `admin@qazaqstay.kz` / `admin123`.
